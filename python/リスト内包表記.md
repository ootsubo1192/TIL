# List Comprehensions

以下のリストに`.png`をつけたいとします。

`color = ['red', 'blue', 'green', 'yellow', 'white']`

```python
for color in colors:
  color_png = color + 'png'
  color_png.append(color_png)
```

`colors_png = ['red.png', 'blue.ong', 'green.png', 'yellow.ong', 'white.png']`
が出力される

これをリスト内包表記で書く

```python
[ color + '.png' for color in colors]
```
